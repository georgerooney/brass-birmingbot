# Python Module: RL Training & Evaluation

This directory contains the Reinforcement Learning (RL) pipeline for Project Brass, including environment wrappers, model architectures, and training scripts.

## Contents & Behavior

- **`brass_env/`**: A Gym/Gymnasium wrapper that communicates with the Go engine. 
    - **Updated**: Uses high-performance Go C-bindings instead of HTTP to maximize throughput.
    - Handles high-speed tensor decoding.
    - Provides a standard interface for `stable-baselines3`.
- **`train.py`**: The main training script, now focused purely on the training loop execution.
    - Supports an optional **Card Attention Transformer** via the `--use-transformer` flag.
    - Implements non-strict loading to allow loading checkpoints across architecture changes.
- **`config.py`**: Contains all training hyperparameters, network architecture defaults, and curriculum/LR schedulers.
- **`utils.py`**: Contains heavy evaluation logic like `DiagnosticCallback` and `run_diagnostics` to keep training runs clean.
- **`test_agent.py`**: Evaluation script. Generates JSON "traces" including full game states, used by the dashboard for visualization.
- **`runs/`**: Contains TensorBoard logs, checkpoints, and final model weights.

## Neural Architecture: Card Attention Sidecar (Optional)

The architecture supports two modes depending on the `--use-transformer` flag:
1. **Standard MLP**: Direct processing of the concatenated board and hand vectors (the setup currently yielding ~90 VP).
2. **Card Attention Transformer**: 
    - Projects 24-D card vectors to a 16-D latent space.
    - Applies a single-head Transformer encoder to process relationships between the 8 cards in hand.
    - Projects the pooled output back to 192-D to maintain exact dimension compatibility with standard MLP checkpoints.

## Key Linkages

- **Communicates With**: Go engine via direct C-bindings.
- **Outputs To**: `eval_data/` (traces/index) which is consumed by the `dashboard-v2` frontend.
- **Dependencies**: `stable-baselines3`, `sb3-contrib` (MaskablePPO), `torch`, `uv`.
